💬Help & Support

Welcome to the help page. Here you can find detailed information and support for using the Gesture Art application effectively.

🎨Getting Started

To begin painting, please register or log in. Once logged in, access the painting page where you can use hand gestures to create stunning digital art.

👉How to Use Hand Gestures

  • Open hand: Cursor
  • Fist: Pause drawing or stop brush
  • One finger: By using the index finger U can draw
  • Two fingers: Select the color at the top

📝Features

  • Hand gesture recognition for intuitive painting
  • Undo and redo functionality for easy corrections
  • Save and load your drawings seamlessly
  • Integrated webcam feed for real-time interaction
